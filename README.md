# My Noctalia Plugins

A small collection of custom Noctalia plugins for Linux, starting with a weather indicator plugin.

## Included Plugin

- `weather-indicator-custom`
  - A Noctalia plugin that shows current weather conditions, temperature, and a seven-day forecast panel.
  - Includes a bar widget, Control Center widget, settings page, and translations.

## Plugin Details

The `weather-indicator-custom` plugin contains:

- `BarWidget.qml`
- `ControlCenterWidget.qml`
- `Panel.qml`
- `Settings.qml`
- `WeatherCardExtra.qml`
- `manifest.json`
- `i18n/` translations
- `preview.png`

## Installation

1. Copy the `weather-indicator-custom` folder into your Noctalia plugins directory.
2. Restart Noctalia or refresh plugins.
3. Enable the plugin from Noctalia settings.

## Development

- `manifest.json` defines plugin metadata and entry points.
- `README.md` inside each plugin folder documents plugin-specific details.
